# Лабораторная работа №1  
### *Прямой лексический анализатор (Вариант 1)*

---

## 🎯 **Цель работы**

Доработать **прямой лексический анализатор** для распознавания конструкций языка программирования в стиле **Pascal**.

---

## 📂 **Структура проекта**

```text
dir_scan/
│
├── SCANER.cpp          # Основной модуль распознавания
├── MAIN.cpp            # Точка входа, работа с файлами
├── TESTOUT.cpp         # Вывод результатов
├── error.cpp           # Обработка ошибок
│
├── scaner_type.h       # Типы лексем (lc_type)
├── scaner_local.h      # Таблица ключевых слов
├── scaner_data.h       # Глобальные данные
├── scaner_ext.h        # Внешние функции
│
├── DIR_SCAN            # Исполняемый файл
│
├── test_operators.txt  # Тест операторов
├── test_keywords.txt   # Тест ключевых слов
├── test_numbers.txt    # Тест чисел
├── test_comments.txt   # Тест комментариев
├── test_labels.txt     # Тест меток
├── test_error.txt      # Тест ошибок
│
└── README.md           # Данный файл
